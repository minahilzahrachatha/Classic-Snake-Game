PROJECT DESCRIPTION: The game demonstrates core programming concepts such as event handling, loops, conditional logic, and object-oriented design. The player controls a snake that moves around the screen, eats food, grows in length, and the game ends when the snake hits the wall or itself.

TECHNOLOGIES USED: Programming Language: C# Framework: .NET (Windows Forms) IDE: Visual Studio Platform: Windows

PROJECT STRUCTURE: Program.cs – Application entry point Form1.cs – Main game logic
Form1.Designer.cs – UI design and components

HOW TO RUN THE PROJECT:

Download or clone the repository
Open the .sln file in Visual Studio
Build the solution
Run the project using Start
